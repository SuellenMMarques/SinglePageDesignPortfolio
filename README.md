# 🎨 Single Page Design Portfolio

Este é um projeto de landing page desenvolvido com foco em prática de HTML, CSS e responsividade, baseado em um desafio do Frontend Mentor.

## 🚀 Visão Geral

O objetivo deste projeto foi construir uma página moderna e responsiva, seguindo um design proposto no Figma, com atenção à estrutura semântica, organização de estilos e boas práticas de desenvolvimento front-end.

---

## 🔗 Links

* 💻 **Repositório:** https://suellenmmarques.github.io/SinglePageDesignPortfolio/

---

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3 (Flexbox e Grid)
* JavaScript (funcionalidade do slider)

---

## 🎯 Funcionalidades

* Layout totalmente responsivo
* Estrutura semântica com HTML5
* Slider de projetos funcional
* Botões interativos
* Boas práticas de acessibilidade (uso de aria-label em botões sem texto)

---

## 📚 Aprendizados

Durante o desenvolvimento deste projeto, pratiquei:

* Organização de CSS
* Uso de Grid e Flexbox
* Estruturação de layout responsivo
* Melhoria na semântica HTML
* Introdução à acessibilidade (ARIA)

---

## ⚠️ Desafios

* Ajustar o layout responsivo conforme o design
* Organização do CSS de forma escalável
* Implementação e entendimento do comportamento do slider

---

## 📌 Melhorias Futuras

* Refatoração completa do CSS com metodologia (ex: BEM)
* Separação de arquivos CSS por responsabilidade
* Melhorar acessibilidade geral da aplicação
* Adicionar animações sutis para melhorar a experiência
